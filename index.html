About Me

I am a web developer who is located right outside of Atlanta, Georgia. My interests include playing video games, sports and spending time with friends, family and my two adorable dogs. Before becoming a web developer, I worked as a licensed social worker. I was initially attracted to the social work field because I enjoy working with people; however, after exploring different social work positions, I discovered that I craved a career that involved innovation and design. I was first introduced to coding through my brother, a Game Design major, who recommended that I learn C#. As I began experimenting with C# and making basic games in Unity, I discovered that I enjoyed both the coding and design process. Just like my favorite choice driven video games, coding is like a puzzle that can be solved in a variety of ways. Spaced Repetition

Github

Spaced Repetition is a responsive web application that allows users to learn a foreign language. The application utilizes an algorithm to implement the spaced repetition learning technique, which allows for more effective learning of a new language. Our application currently allows users to learn Italian, but is structured for the potential to add additional languages in the future.
